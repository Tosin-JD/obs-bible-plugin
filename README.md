# obs-bible-plugin

This is a free bible plugin that you can use to display bible passages.

# obs-bible-plugin

# How to Install the plugin
1. Download the obs-bible-plugin.zip
2. Extract the plugin
3. Copy to a safe location on your computer.
4. Copy the plugin to a safe location in your computer.
5. Copy the URL or PATH of the control_panel.html
6. Click on Dock > Custom Docks > Add a new Dock > Paste the URL of the Control Panel in the URL section and give a name for the Dock
7. Copy the URL or PATH of the browser_source.html
8. Click on Source Add
9. Click on Browser
10. Paste the URL or the PATH in the place for URL
11. Specify the width and height of the Browser
12. Click on okay.
13. You can type on the textbox to display text and you can click on bible passages to display it.
